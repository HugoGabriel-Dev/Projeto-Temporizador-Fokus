## 🍅 Técnica Pomodoro

A técnica Pomodoro é um método de gerenciamento de tempo que alterna períodos de foco e descanso para melhorar a produtividade.

Neste projeto, o temporizador ajuda a organizar o tempo de estudo ou trabalho utilizando ciclos de foco e pausa.
